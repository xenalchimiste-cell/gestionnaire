# 🚀 DevProject Manager

> Une solution moderne et artisanale pour la gestion de projets et de tâches, conçue spécifiquement pour les développeurs.

## 📋 Présentation
DevProject Manager est une application web complète (Fullstack) permettant d'organiser vos projets de développement. Elle utilise une architecture robuste et une interface utilisateur soignée pour offrir une expérience fluide, de la gestion des tâches au suivi des performances globales.

## ✨ Fonctionnalités Clés
- **🔐 Authentification Sécurisée** : Système de connexion et d'inscription pour protéger vos données.
- **📂 Gestion de Projets** : Créez des espaces dédiés pour chaque projet avec des codes couleurs personnalisés.
- **📋 Tableau Kanban** : Gérez vos tâches selon trois états : *À faire*, *En cours* et *Terminé*.
- **📊 Dashboard Personnel** : Visualisez vos statistiques (nombre de projets, taux de complétion, priorité haute).
- **⚡ Activité Récente** : Gardez un œil sur les dernières modifications effectuées.
- **🛡️ Panel Administration** : Une vue d'ensemble pour les administrateurs avec export des données au format CSV.

## 🛠️ Stack Technique
- **Frontend** : HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Backend** : Node.js, Express
- **Base de données** : SQLite (persistance locale)
- **Design** : Variables CSS, Polices Google (Inter, Bricolage Grotesque)

## 🚀 Installation & Lancement

1. **Cloner le projet**
   ```bash
   git clone [url-du-repo]
   cd "projet slam"
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur**
   ```bash
   npm start
   ```
   L'application sera accessible sur `http://localhost:3000`.

## 📂 Structure du Projet
- `server.js` : Point d'entrée du backend Express.
- `database.js` : Configuration et initialisation de SQLite.
- `app.js` : Logique frontend de l'application.
- `index.html` : Structure de l'interface.
- `style.css` : Design system et styles personnalisés.
